# 🐾 Pawsitive Friends

A web-based platform designed to connect people with pets in need of care, companionship, and safe homes. Users can explore, adopt, foster, or sponsor pets while supporting shelters in meaningful ways.

## Features

- User account creation and authentication  
- Browse available pets  
- View detailed pet profiles  
- Adopt, foster, or sponsor animals  
- Admin management of pets (add, edit, delete)  
- Responsive design for all devices  
- Interactive UI with animations  

---

## Project Overview

Pawsitive Friends focuses on creating an accessible and flexible platform for animal lovers. It allows users to engage with pets in different ways, whether through adoption, temporary care, or general support.

---

## Tech Stack

- **Frontend:** Blazor (Razor Components), HTML, CSS, Bootstrap  
- **Backend:** .NET  
- **Database:** Azure SQL / SQL Server (Entity Framework Core)  
- **Authentication:** .NET Identity, Google OAuth  
- **Hosting:** Azure / Render
